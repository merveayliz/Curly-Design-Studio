# 🎨 Curly Design Studio | Concept Project

Welcome to **Curly Design Studio**, a modern, responsive, and interactive landing page concept. This project was developed as a key milestone in my journey to becoming a Full-Stack Developer, focusing on mastering advanced Frontend techniques and JavaScript logic.

## 🚀 Overview
Curly Design Studio is a showcase of how clean design meets functional code. The primary goal of this project was to move away from static HTML and implement dynamic content rendering using JavaScript.

🔗 **Live Demo:** [Click here to view the project](https://merveayliz.github.io/Curly-Design-Studio/)

---

## ✨ Key Features

### 1. Dynamic Content Rendering (JS Logic)
Instead of hardcoding every product and testimonial, I utilized **JavaScript Objects and Arrays**. This allows the site to be easily scalable and demonstrates my understanding of data structures.
* **Automated Grid:** Products are rendered dynamically using `.map()` and template literals.

### 2. Modern UI/UX Design
* **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop using **CSS Grid** and **Flexbox**.
* **Interactive Elements:** Engaging hover effects and smooth scroll animations to enhance user retention.
* **Glassmorphism Effects:** Modern UI trends implemented with `backdrop-filter` for a premium feel.

### 3. Clean Code & Performance
* Semantic HTML5 for better SEO and accessibility.
* Optimized CSS variables (`:root`) for easy theme management and consistency.

---

## 🛠 Tech Stack

* **HTML5:** Semantic structure.
* **CSS3:** Custom animations, Flexbox, Grid, and responsive design.
* **JavaScript (ES6+):** DOM manipulation, dynamic rendering, and event handling.
* **GitHub Pages:** Deployment and hosting.
